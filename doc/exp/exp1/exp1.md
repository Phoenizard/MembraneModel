# exp1 — Fig.2 收敛性测试（两组分囊泡）

> 方法与全部公式以 [`../../note/IEQ_scheme_vesicle.md`](../../note/IEQ_scheme_vesicle.md) 为准；本文件只记录本实验的参数取值与实测结果。

## 模型参数

| 符号 | 含义 | 值 |
|---|---|---|
| Ω | 计算区域 | `[−π, π]³`，周期边界（盒体积 `(2π)³≈248`） |
| `ε = ξ` | 界面宽度 | `0.1964` |
| `k` | 弯曲刚度 | `1` |
| `c` | 两相刚度差 | `0`（⇒ `k(η) ≡ 1`，单一刚度） |
| `δ` | 线张力 | `10` |
| `v_d` | 体积差 `∫φ` | `−216.52` |
| `a_0` | 总面积 | `29.46` |
| `a_d` | 面积差 | `0.23` |
| `c₀,c₁,c₂` | 自发曲率 | `0` |

## 初始化

- `φ⁰ = tanh((R − |x|)/(√2·ε))`，半径由体积约束反推：`|Ω_i| = (|Ω| + v_d)/2`，`R = (3|Ω_i|/4π)^{1/3} ≈ 1.55`。
- `η⁰ = tanh(z/(√2·ε))`（上下两相，`D ≈ 0`）。

## 实验矩阵

| run | N | `h = 2π/N` | `τ` | 状态 |
|---|---|---|---|---|
| exp1-1-N32 | 32³ | `0.1963` | `1e-3` | ✓ 验证机器 |
| exp1-1-N64 | 64³ | `0.0982` | `2e-4` | ✓ 最终稳定收敛 |

> 64³ 更刚（显式刚性项谱符号 ∝`|k|²`，`|k|²ₘₐₓ` 是 32³ 的 4 倍），故 `τ` 须从 `1e-3` 降到 `2e-4`（`τ≥5e-4` 在 64³ 直接 NaN）。

## 结果（实测）

| run | E_M | E_bend | E_line | V−v_d | A−a_0 | D−a_d | N | P |
|---|---|---|---|---|---|---|---|---|
| N32 | `92.50` | `31.99` | `57.21` | `−2.6e-4` | `+2.9e-3` | `−8.1e-6` | `2.5e-2` | `6.7e-3` |
| N64 | `90.67` | `30.75` | `59.55` | `−1.2e-4` | `+7.4e-4` | `+7.3e-7` | `8.4e-3` | `1.6e-3` |

- **N64**：单档 `M=1e4` 即 `conv=True`（4850 步），约束残差全 `< outer_tol`，无 NaN。
- **N32**：验证跑（旧 `|d|` 判据，取 `M=1e4` 档值）；32³ 欠解析（`ε/h≈1`），`M=1e4` 时 `A` 残差 `2.9e-3` 未达 `outer_tol` → 故 **64³ 才是有效分辨率**。
- `E_M` 两分辨率均落 **~90 量级、跨分辨率一致** → 代码正确，**exp1 通过**。
