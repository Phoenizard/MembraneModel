# exp1 — IEQ 方案首测：Fig. 2 收敛性测试

## 概要

本实验是 [IEQ 一阶解耦数值方案](../../note/IEQ_scheme_vesicle.md) 的**第一个跑通目标**，对应论文
Wang & Du (2008) 的 **Fig. 2 收敛性测试**（参数见 [section3 实验清单](../../note/section3_experiments_zh.md) 的 Fig. 2 条目）。

选它作首测的理由（详见对话记录）：

1. **参数完整**，无缺值；
2. **有定量参照能量**（124.49 / 123.82），是 Sec.3 中唯一能直接对数的 case；
3. `a_d=0.23` 极小（相对 `a_0=29.46`），两相几乎不分离 → 可设 `c=0` 关闭两相对比，
   让 `L,D,N,P` 几乎失活，**先单独验证 φ 侧机器**（弯曲项 `q`、面积 `A`、体积 `V`、
   谱算子 LHS 的 SPD 性、`C₀/σ` 标定），再逐步加复杂度。

> 方法论与全部公式以 [`IEQ_scheme_vesicle.md`](../../note/IEQ_scheme_vesicle.md) 为准，本文件只记录
> **本实验的具体参数取值、配置选择与验收标准**。

## 物理 / 模型参数

通用设定（论文 Sec.3 默认，除非下表覆盖）：

| 项 | 值 | 说明 |
|---|---|---|
| 计算区域 Ω | `[-π, π]³` | 周期边界，盒体积 `(2π)³ = 248.050` |
| 自发曲率 | `c₁ = c₂ = 0` ⇒ `c₀(η) ≡ 0` | 关闭自发曲率 |
| 界面宽度关系 | `ξ = ε` | |
| 求解方式 | `L²` 梯度流演化至稳态（公式 16） | 谱方法 + FFT |
| 线张力 `δ` | `10.0` | Fig.2 中物理上与测试目的无关，取 zh 文档默认值 |

约束目标值（Fig. 2 全部子实验共用）：

| 符号 | 值 | 含义 |
|---|---|---|
| `v_d` | `-216.52` | 体积差 `∫φ dx = |Ω_i| - |Ω_e|`；负值正常（盒内膜外空间更大） |
| `a_0` | `29.46` | 总面积约束 `A = ∫W(φ) dx` |
| `a_d` | `0.23` | 面积差约束 `D = ∫tanh(η/ξ)·W(φ) dx`（极小） |

弯曲刚度（本实验自定，**非论文原值**）：

| 符号 | 值 | 说明 |
|---|---|---|
| `k` | `1.0` | `k(η) = k + c·tanh(η/ξ)` |
| `c` | `0.0` | **置零** → `k(η) ≡ 1`，单组分；满足 `k > |c|`，`√(k/ε)` 有定义 |

> 取 `c=0` 是首测的关键简化：`k(η)` 退化为常数，两相 `η` 不再影响弯曲能，
> `q = √(1/ε)(εΔφ + g(φ))`。第二阶段（见下）再开 `c≠0` 做真正的两组分验证。

## 数值方案配置（IEQ，路线 B「完整 IEQ」）

完整推导见 [`IEQ_scheme_vesicle.md`](../../note/IEQ_scheme_vesicle.md) 第 5 章。本实验的配置选择：

### 辅助变量（第 2 节）

| 变量 | 定义 | 服务对象 |
|---|---|---|
| `q` | `√(k(η)/ε)·(εΔφ + g(φ))` | 弯曲能 `E = ½‖q‖²` |
| `r` | `√(2W(φ) + C₀)` | 线张力 `L`、面积 `A`、面积差 `D`（共用） |
| `s` | `√ε·(∇φ·∇η)` | 正交正则 `N = ½‖s‖²` |
| `p` | `(ξ/2)|∇η|² − (1/4ξ)((η²−1)²)` | η 剖面正则 `P = ‖p‖²` |

其中 `g(φ) = (1/ε)φ(1−φ²)`，`W(φ) = (ε/2)|∇φ|² + (1/4ε)(φ²−1)²`。

> `c=0` 下 `q` 的系数 `√(k/ε)` 为常数；`a_d` 极小 ⇒ `D,N,P` 贡献微弱，但格式仍按完整 IEQ 组装（不特判）。

### 时间格式（第 5 章设计决策）

- **解耦交替**（D3）：每步 `(φⁿ,ηⁿ) → φ-子步(固定ηⁿ) → φⁿ⁺¹ → η-子步(固定φⁿ⁺¹) → ηⁿ⁺¹`。
- **辅助变量隐式回代**（D1/D2）：`q,r,s,p` 取 `n+1` 层，由演化恒等式线性回代消元；
  非线性系数因子（`a(η),g'(φ),1/r,Φ_η,tanh,W` 等）冻结到旧层。
- **罚项外层 Newton 线性化**（D4）：仅半正定 Hessian 主项 `M·dG⊗dG`（秩一）进 LHS，
  含可变号 `(G−g_*)` 的二阶项显式入右端 → LHS 严格 SPD。
- **双井项 Eyre 凸劈分 + 稳定化**（D5）：`σ(·ⁿ⁺¹−·ⁿ)`，σ>0。
- **谱空间**（5.6）：变系数算子劈成「常系数隐式（谱对角）+ 变系数余项显式入右端」；
  秩一罚项算子用 Sherman–Morrison / Woodbury 在谱空间低秩修正；体积约束仅作用零波数。

### 待标定数值参数（5.8，本实验需扫定）

| 参数 | 含义 | 初始建议 |
|---|---|---|
| `C₀` | `r=√(2W+C₀)` 的正则常数，保证 `1/rⁿ ≥ 1/√C₀` 不奇异 | `~1e-2`，扫 `[1e-3, 1e-1]` |
| `σ` | Eyre 稳定化常数；η-子步 `B_L` 负零阶项亦靠它压住 | 由 `δ/ξ·max W` 量级估，扫定 |
| `τ` | 时间步 | 从 `1e-4` 起，按稳定性放大 |
| `ā²` 等 | 常系数劈分代表值（均值 / 上界） | 先取空间均值 |
| `ρ` | 罚因子 continuation 放大比 | `2`–`10`；`M` 从小值起，收敛后 ×ρ 重启 |
| `tol` | 稳态判据 `‖δφ‖+‖δη‖ < tol` | `~1e-6`（先松后紧） |

> 罚因子 continuation（5.7）：`M` 从小（如 `1e4`）起，每次收敛后 `M ← ρM` 放大并重启，
> 直至约束残差 `|V−v_d|, |A−a_0|, |D−a_d|` 与 `N,P` 均 < 容差。
> 2a 的目标 `M=3.2e5` 即 continuation 的终值；2b 的 `M=1e4` 可作起步档对照。

## 初始条件

膜为闭合球面（vesicle），零水平集 `Γ = {φ=0}`。

```
φ⁰(x) = tanh( (R − |x|) / (√2 · ε) )      # φ=+1 在球内(|x|<R)，−1 在球外
η⁰(x) = tanh( z / (√2 · ε) )              # 上下半球分两相；c=0 时其实无关紧要
```

初始半径 `R` 由约束粗调：盒体积 `(2π)³ = 248.05`，`∫φ ≈ |Ω_i| − |Ω_e| = v_d = −216.52`
⇒ `|Ω_i| ≈ 15.77`。球 `(4/3)πR³ = 15.77 ⇒ R ≈ 1.55`。

> 精确 `R` 不关键（约束由罚项强制收敛）；取 `R ≈ 1.55` 使初始 `A,V` 接近目标即可，
> 实际可在第一步打印 `A⁰, V⁰` 与 `(a_0, v_d)` 比对后微调。
> `c=0` 时 `η⁰` 取何值都几乎不影响演化，可直接取常值或上式。

## 分段说明：Stage A（φ-only）vs Stage B（完整两组分）

代码分两段部署（见 `experiments/exp1/run.py`，求解器在 `src/membrane/`）：

- **Stage A**：`c=0` 单组分，能量仅 `E + A + V`（忽略 `η/L/D/N/P`）。先把最难的数值机器
  （6 阶 IEQ 弯曲算子、谱劈分、Woodbury 秩一修正、罚因子 continuation）跑稳。
- **Stage B**：加 η 与 `L,D,N,P`，复现完整两组分问题。

**关键事实（论文 p.355 核实）**：Fig 2 总能量 `124.42 = 弯曲能 47.96 + 线张力能 76.46`，
**线张力占大头**。故 `124.49/123.82` 是 **Stage B** 的验收目标，**Stage A 不可能复现**
（Stage A 没有线张力）。Table 3 亦佐证总能量含 `E_l`。

## 验收标准

### Stage A（本段，已通过）

1. **数值健康**：全程无 NaN/Inf；`rⁿ ≥ √C₀`；`L̂(k) > 0` 对所有 k。✅（`r_min≈0.10`）
2. **正确性门**：`pytest tests/` —— 谱算子机器精度 + 变分导数有限差分校验 + Woodbury 求解残差。✅（7 passed）
3. **约束满足**：continuation 终态 `|V−v_d|≈1.5e-4, |A−a_0|≈2.5e-4`（< 1e-3，量级同 Table 1）。✅
4. **能量单调**：固定 M 段内 `E_M` 单调下降（32033→34.49，无正跳变）。✅
5. **自洽**：网格 `48³ vs 64³` → `34.4878 vs 34.4877`（5 位吻合）；ε 改变给出合理的弥散界面敏感性。✅

> Stage A 稳态弯曲能 `E_b≈34.5`(64³,ε=.1964) 是**单组分**值，与论文两组分的 `48` 无可比性——
> 这正是 Stage B 才能对的数。

### Stage B（含 η/线张力，已实现并验证）

代码：`src/membrane/model/full.py`（完整 `E_M(φ,η)` + `δE_M/δφ, δE_M/δη`）、
`src/membrane/solver/coupled.py`（φ/η 解耦预条件步进 + continuation）、
`experiments/exp1/run_stageB.py`。

**已通过：**
1. **正确性门**：`tests/test_full_grad.py` —— `δE_M/δφ` 与 `δE_M/δη` 的**逐项**有限差分校验
   （弯曲含 `k(η)`、L、A、V、D、N、P 各项 + 全开），16 tests。✅（合计 23 passed）
2. **方法论一致**：论文 p.353 用半隐式 Euler（线性部分隐式、非线性显式，算子劈分）+ 自适应 Δt；
   本实现的预条件梯度步进与之等价。✅
3. **稳定 + 单调**：σ=2、τ=1e-3 全程稳定、能量单调下降、约束/正则全部满足
   （`|V−v_d|,|A−a_0|~1e-4`，`D−a_d~1e-6`，`N,P~1e-2`）。✅
   （σ=0、τ≥2e-3 会发散——σ/τ 为 5.8 待标定量，已设安全默认。）

**关于 124.49（重要，尚未对上）：** 论文 p.354 揭示 Fig 2 的初值**不是带约束的球**，而是
「球 + η 两相 → 关体积约束 `M1=0`、线张力驱动成长椭球/葫芦 → 中途停 → 再加回体积约束」
得到的中间形状；梯度流"收敛到**稳定局部极小**"。即 **124.49 是葫芦分支上的一个路径依赖局部极小**。
此外 Fig 2 的 `k`、`δ` 论文/笔记均未给（zh 注明 δ 对收敛测试目的无关、任取）。

本实现以 `k=1, c=0, δ=10` 复现时：稳态 `E_M≈90–110`（视路径），`E_b≈30, E_l≈60–80`——
显著低于 124.49，且分解与论文（48/76）不符。**结论：要对上 124.49，需要论文未给的 `k`/`δ`
+ 葫芦分支初值协议**，属物理参数辨识（开放问题），非代码正确性问题。

> **建议**：把定量复现的重心放到 **exp2 = Fig 3**（参数完整：`k=2,c=1`→`k₁=3,k₂=1`、`δ=10`、
> `v_d,a_0,a_d` 全给），那是可严格对形态的两组分实验；exp1 Stage B 以"机器正确 + 收敛性质"收尾。
