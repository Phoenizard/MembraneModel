# 第 3 节实验清单 — 两组分囊泡
*Wang & Du (2008), "Modelling and simulations of multi-component lipid membranes and open membranes via diffuse interface approaches"*

## 优化问题回顾

- **目标函数:** 最小化带惩罚项的相场能量 `E_M(phi, eta)`(公式 15)
- **决策变量:** 场函数对 `phi(x)`、`eta(x)`
- **物理约束:** `v_d`(体积差)、`a_0`(总面积)、`a_d`(面积差)
- **正则项(非物理约束):** `N(.,.)` 正交性、`P(.)` 维持 eta 的 tanh 剖面
- **模型参数:** `eps, xi`(扩散界面宽度);`delta`(线张力);`c0 -> (c1, c2)`(自发曲率);`k -> (k, c)` 给出两相刚度 `k1=k+c`、`k2=k-c`
- **区域:** Omega = [-pi, pi]^3,位于 R^3

## 通用设定(除非特别说明)

- 区域 Omega = [-pi, pi]^3,周期边界条件
- c1 = c2 = 0(关闭自发曲率 —— 隔离变量,使复杂形态只能归因于刚度差异 + 线张力)
- xi = eps
- 用梯度流求解(公式 16);谱方法 + FFT
- 第 3 节默认分辨率:eps = 0.1736,64^3 网格
- 网格间距 h:64^3 网格时 h = 2*pi/64 ≈ 0.0982;eps 常以 h 的倍数表示

## 复现注意事项

1. **缺失参数:** 许多实验未给 eps、M_i 或部分约束值。按文中说明,除非特别声明,使用 eps = 0.1736、64^3 网格;其余缺失项沿用此默认。
2. **初始形状链:** 从 Fig. 3 起,多数初始形状来自前一实验的结果改造。链条大致为:Fig. 1 -> Fig. 2 / Fig. 3 -> Fig. 4 -> ...。复现时初始形状必须沿此链条逐级生成。
3. **符号约定:** v_d 是体积差 |Omega_i| - |Omega_e|;取负值是正常的,因为盒子内膜外部空间更大。
4. **纯数值测试中的 delta:** 对 Fig. 2 / Table 1-2,线张力取值不影响测试目的,任取一个一致的值(如 delta = 10)即可。
